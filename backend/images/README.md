# Product Images

Add your custom product images here.

Recommended size: 300x200 pixels
Supported formats: .jpg, .jpeg, .png, .gif

Example files:
- cheddar-cheese.jpg
- chicken-breast.jpg
- tomatoes.jpg
- carrots.jpg

Then update the image URLs in simple_app.py to:
- http://localhost:5000/images/cheddar-cheese.jpg
- http://localhost:5000/images/chicken-breast.jpg
- etc.
